## Debug
해당 프로젝트에서는 환경 변수 파일을 사용합니다. 해당 프로젝트를 디버그하기 위해서 `db.conf` 파일을 생성한 후 시도하세요.

> `db.conf`
> ```
> [DB 호스트명]
> [DB 접속 사용자]
> [DB 접속 사용자 비밀번호]
> [접속 DB명]
> [접속 포트]
> ```

> `Example`
> ```
> localhost
> root
> 1234
> mysql
> 3306
> ```
